DataProtector.jk

DataProtector.jk (just kidding) is a website that exploits a user's disregard of security measures or incapability to recognize an unsafe website. DataProtector.jk uses a common online scam called "phishing" to send emails disguised as an authentic website (e.g. Facebook, Wells Fargo, etc.) to gain a user's login information. DataProtector.jk makes use of a user's onboard microphone to literally and metaphorically collect the user's data then output that data into visible changes similar to how a user might notice a small/large change in their bank account balance.
